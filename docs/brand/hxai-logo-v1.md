# HXAI Logo 基准稿 v1

## 文件

- 基准预览图：`public/brand/hxai-logo-concept-v1.png`
- 画布：1254 × 1254 px
- 当前用途：设计母版，尚未替换网站正式 Logo

## 字形结构

1. 字母顺序固定为 `HXAI`。
2. `H` 与 `X` 重叠，但不是完全融合：
   - H 的横线右端为箭头形尖角。
   - H 横线与 X 左侧尖角之间保留一条清晰的深色缝隙。
   - X 的左上、左下两条青色斜线保持完整。
3. X 的右侧斜笔与 A 的左下区域发生视觉叠接。
4. `AI` 必须作为连续字形：
   - A 的右侧斜笔自然连接到 I 的竖笔。
   - A 与 I 不得拆成两个独立字母。
5. I 顶部使用独立圆点：
   - 圆点略宽于 I 的竖笔。
   - 圆点与竖笔之间保留小段深色间距。
6. 所有主笔画粗细接近，端点使用平切或斜切，不使用圆角。

## 基准配色

| 部位 | 颜色 | 用途 |
| --- | --- | --- |
| 背景 | `#05070D` | 网站主背景 |
| H 与 X 左侧斜笔 | `#67E8F9` | 主品牌青色 |
| X 右侧斜笔、A 横线 | `#FB7185` | 对比强调色 |
| A 与 I 主体 | `#86EFAC` | 价格友好、可用状态 |
| I 顶部圆点 | `#FBBF24` | 数据节点、提醒状态 |

当前 PNG 带有轻微明暗变化。制作正式 SVG 时优先使用上述纯色，并另行验证纯色版与
轻微渐变版在导航栏、移动端和 favicon 中的效果。

## 不可变规则

- 不改变 `HXAI` 的字母顺序。
- 不拆开 AI。
- 不删除 I 顶部圆点。
- 不让 H 横线与 X 连死。
- 不在 HX 交叉处加入灰色块。
- 不加入芯片、机器人、脑图、星光或电路装饰。
- 不使用阴影、立体挤出和复杂发光效果。

## 后续可调整项

- X 右侧斜笔与 A 横线的强调色。
- I 圆点颜色与尺寸，调整幅度不超过当前视觉尺寸的 15%。
- H 与 X 之间缝隙宽度，调整时必须保持 24 px 小尺寸下可见。
- 整体横向压缩比例和导航栏适配尺寸。
- 纯色版、轻微渐变版和透明背景版。
