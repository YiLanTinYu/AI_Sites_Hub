# AI 大模型导航

国内外主流 AI 大模型、Agent、搜索、编程与多模态工具导航页。

## 本地预览

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000/index.html
```

## 链接检查

```bash
npm run check:links
```

检查报告会生成到 `reports/link-check.md` 和 `reports/link-check.json`。

## 新 AI 候选发现

```bash
npm run discover:ai
```

候选报告会生成到 `reports/new-ai-candidates.md` 和 `reports/new-ai-shortlist.md`。
